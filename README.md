# FinancialPlanner

Offline yearly financial planner for iOS (SwiftUI + SwiftData), built from the
design in [`design_handoff_financial_planner/README.md`](design_handoff_financial_planner/README.md).
All data is on-device; no backend, no account. Currency AED, year 2026.

## Status

**Lane 2 — Plan the year** is implemented end-to-end:

| Screen | View | Notes |
|---|---|---|
| B1 · Year Plan | `YearPlanView` | 12-month table, green summary card, "Copy to all months" |
| B2 · Month Plan editor ★ | `MonthPlanEditorView` | live planned-savings + allocation meter, draggable category sliders, over-allocation warning |
| B3 · Plan vs Actual | `PlanVsActualView` | per-category actual/plan bars from transactions, ON TRACK/OVER status |
| B4 · Savings goals | `SavingsGoalsView` | progress ring, pace note, goal list |

Lanes 1 (Set up), 3 (Track) and 4 (Review) are not built yet. The app currently
launches into the Year Plan; B2 is reached by tapping a month, B3 from inside B2,
and B4 from the "Goals" toolbar button.

## Build & run

The `.xcodeproj` is generated by [XcodeGen](https://github.com/yonyz/XcodeGen):

```bash
brew install xcodegen        # if needed
xcodegen generate            # regenerate FinancialPlanner.xcodeproj after adding files
open FinancialPlanner.xcodeproj
```

Then select an iPhone simulator and run. Or from the command line:

```bash
xcodebuild -project FinancialPlanner.xcodeproj -scheme FinancialPlanner \
  -sdk iphonesimulator -destination 'generic/platform=iOS Simulator' build
```

> Regenerate the project with `xcodegen generate` whenever you add or remove
> source files. Min deployment target: iOS 17 (required by SwiftData).

## Notes / deviations

- **Fonts:** the handoff calls for Hanken Grotesk + IBM Plex Mono. SF Pro (system)
  is used as the explicitly-permitted substitute, routed through `Font.ui(...)` /
  `Font.mono(...)` so bundled fonts can be dropped in by editing only `Theme.swift`.
- **B2 sliders** fill relative to the largest category in the month (matching the
  mock's 100/48/30/… proportions). A single category can't be dragged above the
  current top category via the slider in v1.
- Seed data reproduces the handoff numbers exactly: year totals 230,000 / 161,300 /
  68,700 (115% of a 60,000 goal), March's B2 breakdown, and June's B3 actuals.
